# Meta-Learning (LESSONS)

- **Erro**: Vite versão nova incompatível com Node antigo.
- **Solução**: Downgrade para Vite 5.4.0. Sempre verificar versão do Node antes de instalar dependências.
- **Eficácia do Planejamento**: Projetos iniciados com um plano de arquitetura claro (Artifact) resultam em menos erros de build e menor desperdício de tokens do que a codificação direta (YOLO mode). Manter a abordagem Architect First, Coder Second.
